# GraphQL Query Examples for E-commerce API

## Query all categories
\```
{
  categories {
    id
    name
    description
  }
}
\```

## Query all products
\```
{
  products {
    id
    name
    price
    description
    category {
      name
    }
  }
}
\```

## Query a product by ID
\```
{
  productById(id: "1") {
    id
    name
    price
    description
    category {
      name
    }
  }
}
\```

## Query products by category ID
\```
{
  productsByCategory(id: "1") {
    id
    name
    price
    description
  }
}
\```

## Query a category by ID
\```
{
  categoryById(id: "1") {
    id
    name
    description
  }
}
\```

## Query a category by name
\```
{
  categoryByName(name: "Electronics") {
    id
    name
    description
  }
}
\```

## Query products by category name
\```
{
  productByCategoryName(name: "Electronics") {
    id
    name
    price
    description
  }
}
\```
